\# BooksVersusMovies Automation



Automated pipeline for creating and upgrading book vs film comparison pages on \[booksversusmovies.com](https://booksversusmovies.com).



\## What This Project Does



\- \*\*Creates new pages\*\* for upcoming 2026 adaptations (\*Verity\*, \*Hunger Games Sunrise\*, etc.)

\- \*\*Upgrades existing pages\*\* to the site’s “gold standard” (cast table, FAQ, SEO meta tags, last‑updated date)

\- \*\*Batch processes\*\* 175+ HTML files using LLMs (Claude, DeepSeek) via OpenRouter

\- \*\*Preserves affiliate links\*\* and existing structure while injecting new content sections
