# DeepSeek V3 – Practical Advice

## 1. Business Advice (Revenue, SEO, Growth)

**SEO leverage**  
- Generate **meta descriptions** with high CTR potential by testing variants like:  
  - "Book vs Film: 5 Key Differences in [Title]"  
  - "Why the [Title] Film Changed the Ending (Spoiler-Free Comparison)"  
- Build **internal links** using keyword-rich anchors:  
  - "See how *The Shining* adaptation diverges" instead of "More here"  
- Automatically create **FAQ schema** for voice search:  
  - "Is the book better than the movie?"  
  - "What chapters were cut from the film?"  

**Backlink opportunities**  
- Produce **stat-driven comparison tables** (e.g., "Page Count vs Runtime Analysis") for data journalism links  
- Generate **authoritative adaptation timelines** (e.g., "Every Stephen King Film Adaptation Ranked by Publication Date")  

**Audience growth**  
- Create **TikTok/Reels scripts** from key differences:  
  - "3 scenes the book did better #BookTok"  
- Auto-generate **Reddit discussion prompts**:  
  - "DAE think the film's casting ruined [character]?"  

**Monetization**  
- Inject **affiliate CTAs** naturally:  
  - "Prefer the original story? [Get the book here](#affiliate-link) before watching."  
- Generate **comparison widgets** with "Read/Watch" toggle buttons driving conversions  

**Competitive edge**  
- Mass-produce **"Director's Cut" analyses** comparing theatrical vs extended editions against the book  

## 2. Technical Advice (Prompts, Workflow, Output)

**Prompting best practices**  
- System message:  
  ```  
  You are a book-to-film adaptation analyst. Output JSON with:  
  - verdict (string)  
  - differences (array{heading,text})  
  - fidelity_score (1-10)  
  ```  
- Temperature: 0.2 for factual comparisons, 0.6 for opinion pieces  

**Affiliate workarounds**  
- Never prompt with HTML containing links – inject post-generation  
- For CTAs: "Output placeholder [LINK] for later substitution"  

**Token efficiency**  
- Typical page: ~800 tokens (JSON) vs ~4k tokens (HTML)  
- Cost estimate: $0.003 per comparison page  

**Integration patterns**  
1. Extract existing page sections for context  
2. Parallel API calls for each component (meta, cast, differences)  
3. Cheerio injection with atomic updates  

**Known issues**  
- May hallucinate minor characters – validate against IMDb data  
- Sometimes over-scores faithfulness – add manual weightings  
- Requires strict JSON validation (trailing commas common)
